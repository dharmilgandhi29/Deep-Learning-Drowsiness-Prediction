# Deep-Learning-Drowsiness-Prediction

## Datasets

We use two public datasets:

1. **MRL Eye Dataset (Infrared)**  
   Kaggle: https://www.kaggle.com/datasets/akashshingha850/mrl-eye-dataset  
   After downloading and unzipping, place the folder so the structure is:

   Deep-Learning-Drowsiness-Prediction/
     Data/
       MRL/
         data/
           train/
           val/
           test/
           ...

2. **Eye Dataset (Open / Closed Eyes)**  
   Kaggle: https://www.kaggle.com/datasets/prasadvpatil/eye-dataset  
   After unzipping, place the folder so the structure is:

   Deep-Learning-Drowsiness-Prediction/
     Data/
       Eye dataset/
         train_dataset/
           openLeftEyes/
           openRightEyes/
           closedLeftEyes/
           closedRightEyes/
